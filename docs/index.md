## Быстрый старт

| Команда | Назначение |
| --- | --- |
| `npm run dev` | Запуск анимации (Vite) |
| `npm run docs:dev` | Локальный сервер документации |
| `npm run docs:build` | Сборка статического сайта документации |

Документация описывает код в `src/main.ts`: архитектуру `Engine`, физику столкновений и цикл `requestAnimationFrame`.
